# LLM & AI Interview Questions [Hard]

## Question 1: Transformer Architecture

What is the time complexity of self-attention mechanism?

- A) O(n)
- B) O(n log n)
- C) O(n²)
- D) O(2ⁿ)

**Answer:** C

## Question 2: Attention Mechanism

In the attention formula `Attention(Q,K,V) = softmax(QK^T/√d)V`, what is the purpose of √d?

- A) Speed up computation
- B) Prevent vanishing gradients
- C) Scale the dot products to control variance
- D) Regularization

**Answer:** C

## Question 3: RAG System

What is the main advantage of RAG over pure fine-tuning?

- A) Faster inference
- B) Lower cost
- C) Can access up-to-date information without retraining
- D) Better for all tasks

**Answer:** C

## Question 4: Prompt Engineering

Which technique involves adding "Let's think step by step" to the prompt?

- A) Few-shot learning
- B) Zero-shot learning
- C) Chain-of-Thought (CoT)
- D) Constitutional AI

**Answer:** C

## Question 5: Model Training

What is "alignment" in the context of LLM training?

- A) Ensuring model outputs are properly formatted
- B) Making model behavior match human preferences and values
- C) Aligning model weights across distributed training
- D) Text alignment in translation models

**Answer:** B

## Question 6: Embeddings

What is the purpose of positional encoding in transformers?

- A) Compress the input
- B) Add position information to tokens since attention is order-invariant
- C) Reduce model size
- D) Enable parallel processing

**Answer:** B
